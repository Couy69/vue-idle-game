# 挂机放置类小游戏

> 一个全随机的刷装备刷词条小游戏,游戏链接：[couy.xyz](http://couy.xyz)

# 游戏设计

- 装备分为四个等级：破旧、普通、神器、史诗
- 主角默认每秒回血 2%
- 默认暴击伤害 150%
- 完成副本时可能掉落独特级别(5个词条)的装备（通关后2.5%掉落率，无尽挑战不会掉落）
- 通关后开启无尽挑战

# 本地运行

### 安装依赖

```sh
  npm install
```

### 启动项目

```sh
  npm run serve
```

### 打包项目

```sh
  npm run build
```
